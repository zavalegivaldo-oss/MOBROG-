# Checkout M-Pesa

Sistema de checkout integrado com M-Pesa.

## Funções

- Pagamento direto
- Modal integrado
- Estrutura profissional
- Compatível com GitHub
- Preparado para API oficial

## Como usar

1. Faça upload para GitHub
2. Conecte ao Vercel ou Netlify
3. Edite os dados M-Pesa
4. Publique
